Quick Start

Download

NumBlock is a software for multi-dimensional arrays (nd-arrays) processing and visualization.

You can download here for free.

Download NumBlock

Software is still under development More features will be added in the future. So keep an eye on this project :)

Interface

(1) Canvas

Canvas is your workspace with all the blocks shown here.

Each block is a function with 3 components -- inputs, outputs and parameters.

Build your graph by connecting these blocks.

(2) Components

All available blocks are listed here. Click [+] to add block to the canvas.

(3) Array Preview

You can check the content of an array/image here.

(If array is too large, only a portion of array will be shown in the window.)
